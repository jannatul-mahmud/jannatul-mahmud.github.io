# Jannatul Mahmud — Portfolio

Personal portfolio site. Plain HTML/CSS/JS, no frameworks, no build step. Deploys to GitHub Pages.

## Pages

- `index.html` — home
- `projects.html` — projects listing
- `about.html` — about me
- `blog.html` — blog index
- `contact.html` — contact info
- `project-traffic-light.html` — ESP32 project detail (filled in)
- `project-logisim.html`, `project-bst.html`, etc. — stub pages, fill in later
- `blog-traffic-light-build.html` — first blog post (filled in)

## Files to add

- `resume.pdf` — drop your résumé here. Download buttons pick it up automatically.
- `demo.gif` (or `demo.mp4`) — for the traffic light project page

## Deploy

See `DEPLOY.md` for step-by-step.

## Customize

- **Colors:** edit the `:root` block at the top of `styles.css`
- **Fonts:** import statement at top of `styles.css`
- **Content:** open any `.html` file, find the text, change it

## Tech

- Plain HTML5
- CSS with custom properties
- One file of vanilla JavaScript
- Google Fonts: Playfair Display, Plus Jakarta Sans, Space Mono
